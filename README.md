### les Variables 

```
let a = 5
```
il creer une variable a avec la valeur 5 (mais bon vous avez compris)

il va deviner le type automatiquement 

sinon vous pouvez le precisez avec 

```
let a: int = 5
``` 
et voila le type est specifie
